Motoreign Multi Vendor Marketplace

A complete multi-vendor eCommerce marketplace built for Odoo 19 Enterprise. Enable multiple sellers to manage their own products, orders, and profiles while admins control commissions, approvals, and payments.

Key Features

Seller Registration

Sellers sign up via the website and go through an admin approval workflow before listing products.

Commission Management

Set global or per-seller commission rates. Commissions auto-calculate on order confirmation.

Product Approval

Admin manually or automatically approves seller products before they appear on the shop.

Seller Dashboard

Real-time dashboard showing product, order, payment, and inventory request counts.

Inventory Requests

Sellers submit quantity requests; admins approve to update stock using Odoo's inventory.

Seller Payments

Complete payment workflow from request to validation with auto vendor bill creation.

Ratings & Reviews

Customers rate and review sellers. Reviews can be auto-published or manually approved.

ZATCA Ready

Inherits Odoo 19's full VAT/ZATCA compliance out of the box for Saudi Arabia operations.
